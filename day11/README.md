# adventofcode
Solutions for Advent of Code

## Prerequisites

NodeJS >= 6.11.0

## Install

```
npm i
```

## Day4

Usage:
```
npm start -- -p 2 -f puzzleInput.txt
```

Arguments:
* -p: The part of the day to execute (1 or 2)
* -f: The file containing the route to input
* -r: The route if directly entered instead of using a file
